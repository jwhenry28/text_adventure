Leiper: A Text Based Adventure in Python

My first project is a text adventure game called Leiper. I've always loved Zork: The Underground Empire. However, I've only recently began appreciating it for the coding marvel that it is. The Infocom written parser that operated as the heart of Zork is truly genius, being able to handle some fairly non-intuitive and complex English sentences. I've long wanted to write my own parser, and I finally have been able to get it done. My initial plan was to use the Natural Language Toolkit, but this ended up being too cumbersome and heavyweight for my purposes. I settled on a pre-written dictionary dictated by some simple rules written in Regular Expression and some coding logic. The parser essentially reads in an imperative sentence, extracts the verb/action, all direct objects, all indirect objects, and each object's respective adjectives (if any) before compacting all of this information in a neat data structure and passing it off to the state modifier. The parser is nothing to gloat about, but I will say it is far more elegant and flexible and a tangle of if-else ladders dictating the behavior for each preselected command.
